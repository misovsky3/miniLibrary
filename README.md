# Backend

We will be using [Prisma](https://www.prisma.io/) to handle coupling relational database with our JavaScript code.

# Frontend

We will be using [Bootstrap](https://getbootstrap.com/) library to provide an easy-to-use UI library.

# Dev Environment

We will each be using Visual Studio Code and WebStorm, because we each prefer different dev environments. The system of development, however, remains the same or at least very similar in both environments.
